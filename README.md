## 📋 Sobre o App

O **Siget App** é uma solução mobile desenvolvida para otimizar a gestão de transportadoras, oferecendo acesso rápido e prático às informações e operações do dia a dia diretamente pelo celular.

## 📲 Como Instalar

Siga os passos abaixo para instalar o aplicativo no seu celular Android:

**Passo 1** — Baixe o arquivo APK clicando no botão acima

**Passo 2** — Abra o arquivo baixado no seu celular

**Passo 3** — Se aparecer uma mensagem de segurança, siga os passos:
- Acesse **Configurações** do celular
- Vá em **Segurança** ou **Privacidade**
- Ative a opção **"Instalar aplicativos de fontes desconhecidas"**
- Volte e tente instalar novamente

**Passo 4** — Conclua a instalação e abra o app

> ⚠️ **Atenção:** Essa mensagem de segurança é normal para aplicativos instalados fora da Play Store. O Siget App é seguro e desenvolvido exclusivamente para uso corporativo.
